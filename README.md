# K&R's "The C Programming Language" Exercises' Solutions

## Requirements

* GCC
* Make


## Compilation

To compile all of the exercises:

```bash
cd K_R_Exercises
make all

```

To compile only a certain folder's exercises:

```bash
cd K_R_Exercises
cd ChapterX
make all
```
